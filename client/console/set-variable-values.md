# Set Variable Values

## Functions

| Function |
| :--- |
| `SetBool(string var, bool val)` |
| `SetInt(string var, int val)` |
| `SetFloat(string var, float val)` |
| `SetString(string var, string val)` |

## Usage

```lua
console.SetFloat("sensitivity", 1.2)
```
