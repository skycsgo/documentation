# Get Variable Values

## Functions

| Function | Type |
| :--- | :--- |
| `GetBool(string var)` | `bool` |
| `GetInt(string var)` | `int` |
| `GetFloat(string var)` | `float` |
| `GetString(string var)` | `string` |

## Usage

```lua
local sensitivity = console.GetFloat("sensitivity")
```
